**Security Log Analyzer**

  A full-stack security tool that analyzes logs and detects sensitive data such as emails, passwords, and API keys. It evaluates risk levels and provides actionable insights.

**Features**
    1) Detects sensitive data (Email, Passwords, API Keys)
    2) Risk scoring engine (Low / Medium / High)
    3) Automated action recommendation (ALLOW / BLOCK)
    4) File upload support (TXT, LOG, PDF)
    5) Insights generation
    6) Interactive UI dashboard
    7) Timestamped analysis

**Tech Stack**
Backend
    1) Java
    2) Spring Boot
    3) REST API

Frontend
   1) HTML
   2) CSS
   3) JavaScript (Fetch API)

Libraries
   1) Apache PDFBox (PDF processing)
   2) Swagger (API documentation)
